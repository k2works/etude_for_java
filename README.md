## 構築
仮想マシンのセットアップ
```bash
vagrant up
vagrant ssh
```
コンポーネントのセットアップ
```bash
curl -s "https://get.sdkman.io" | bash
sdk install java
sdk install gradle
gradle init
gradle buildEnvironment

```
